# GitHub Traffic Backend

A NestJS-based backend service for tracking and analyzing GitHub repository traffic.

## Description

This application provides a REST API for storing, processing, and visualizing GitHub repository traffic data.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## API Documentation

Once the application is running, you can access the Swagger documentation at:

```
http://localhost:3000/api
```
